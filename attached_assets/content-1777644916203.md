### Identity

The agent's presence on the open internet: a wallet on-chain, an inbox for service logins, and a domain of its own. You stay in full custody and set the guardrails the agent operates within.

#### Wallet

The agent's on-chain wallet, in your **full custody and control**. You hold every key, set spend policies and guardrails, and define the transaction security the agent operates within. Top up with crypto, credit card, or fiat onramp.

[Read docs →](https://os.virtuals.io/agent-identity/wallet/overview)

#### Email

Dedicated inbox where the agent receives verifications and service notifications.

[Read docs →](https://os.virtuals.io/agent-identity/email/overview)

#### Domain

A web identity the agent owns: discoverable, verifiable, and uniquely its own.
